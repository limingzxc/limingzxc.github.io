我的世界forge模组开发学习资源

玩了这么久的模组开发，今天分享给大家一些优质的学习资源

Youtube: https://www.youtube.com/@ModdingByKaupenjoe

这位大佬的模组教程简单易懂，非常推荐。如果无法看油管的话可以看大佬的博客网站 https://tutorialsbykaupenjoe.net/

Kook: TeaCon(模组开发茶会) https://kook.top/lyGudV

这个聊天服务器里有很多模组开发的大佬，而且还有能查询混淆名的机器人，而且这里也比较活跃，是我在国内找到的最好的模组开发聊天群。

MCmod百科 && github

https://www.mcmod.cn/ && https://github.com

我们在MC mod百科网站里查找Mod的信息非常方便，不论你是开发者还是玩家，都可以找到很多有用信息。

我们以旅行者背包mod为例

这里可以清楚的看到这个mod的作者，支持的版本，还有它的github链接。

很多mod都是开源在github上的，所以我们可以查看这些mod的源代码，并且学习作者的代码。

学习的同时不要忘了给作者一个star哦

minecraft wiki 我的世界维基百科

https://minecraft.fandom.com/zh/wiki/Minecraft_Wiki （国外的网站，访问较慢）

https://wiki.biligame.com/mc/Minecraft_Wiki （b站搭建的镜像网站）

我的世界维基百科对开发也很有用，你可以查询很多游戏数据。
